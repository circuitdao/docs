# User Guide

User guide to Circuit protocol. At the bottom of each page is a list of global state parameters (Statutes) defined on that page. For more information see the [Statutes](../../technical-manual/statutes) page in the Technical Manual, or refer to the [dashboard](https://circuitdao.com/protocol) for the latest Statute values. The tokens relevant to the protocol are XCH, the collateral asset, BYC, the stablecoin issued by the protocol, and CRT, the protocol's governance token.
