# Deployments

A deployment is the initial set of smart coins that was minted on the blockchain when the protocol launched. Given how the coinset model works, all coins that belong to the protocol can be traced backed to the deployment, either by virture of being child coins of protocol coins, or by being tied to the protocol via announcements.
